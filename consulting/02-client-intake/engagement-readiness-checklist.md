# Engagement Readiness Checklist

- [ ] Executive sponsor confirmed.
- [ ] Engagement scope and exclusions approved.
- [ ] Evidence owner assigned.
- [ ] Stakeholder interview list approved.
- [ ] Confidentiality expectations documented.
- [ ] Tool access, screen-share process, or evidence export process approved.
- [ ] Kickoff, interview window, assessment window, and readout scheduled.
- [ ] Decision makers understand advisory nature of assessment.
- [ ] Client confirms no restricted data should be provided unless explicitly approved.
- [ ] Closeout and data handling expectations agreed.
