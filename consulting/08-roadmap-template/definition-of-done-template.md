# Definition of Done Template

- [ ] Process changes documented and communicated.
- [ ] Documentation updated in the approved repository or knowledge base.
- [ ] Software changes implemented where applicable.
- [ ] Testing completed with evidence attached.
- [ ] Metrics baseline and post-change measurement defined.
- [ ] Training or enablement completed for affected roles.
- [ ] Leadership approval recorded for operating model changes.
- [ ] Customer validation completed or explicitly scheduled.
