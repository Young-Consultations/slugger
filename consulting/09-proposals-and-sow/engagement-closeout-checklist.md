# Engagement Closeout Checklist

- [ ] Deliverables accepted.
- [ ] Open actions transferred to client owner.
- [ ] Consultant access removed.
- [ ] Client data returned or deleted according to agreement.
- [ ] Final invoice submitted.
- [ ] Feedback requested.
- [ ] Review or recommendation requested when appropriate.
- [ ] Case-study permission requested only if appropriate.
- [ ] Lessons captured.
