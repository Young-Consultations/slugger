# knowledge/

The knowledge base captures reusable engineering guidance, prompt practices, platform references, and lessons learned.

## Structure

- `engineering/` — design patterns, coding standards, and testing strategy.
- `prompt-engineering/` — prompt design guidance and reusable patterns.
- `architecture/` — architectural references and evolving design guidance.
- `python/` — Python-specific conventions and implementation notes.
- `github/` — GitHub workflows, automation, and collaboration references.
- `security/` — security principles and review guidance.
- `testing/` — cross-cutting testing references.
- `deployment/` — deployment, operations, and release knowledge.
- `lessons-learned/` — retrospectives and execution learnings.
- `decisions/` — decision logs and curated rationale.
