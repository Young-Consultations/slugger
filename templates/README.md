# templates/

This directory stores reusable document templates for common AI-SDLC artifacts.

## Available Templates

- `adr-template.md` — architecture decision record template.
- `vision-template.md` — product vision template.
- `requirements-template.md` — requirements specification template.
- `architecture-template.md` — architecture document template.
- `agent-template.md` — agent documentation template.
