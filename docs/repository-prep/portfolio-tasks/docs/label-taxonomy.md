# Label taxonomy

Recommended labels: `project:slugger`, `project:consulting`, `project:ai-sdlc`, `type:feature`, `type:defect`, `type:task`, `type:research`, `type:decision`, `priority:p0`, `priority:p1`, `priority:p2`, `priority:p3`, `status:ready`, `status:blocked`, `status:in-progress`, `status:review`, `status:done`, `agent:codex`, `agent:chatgpt`, `agent:human`, `chatgpt-task`.

Do not create labels containing GitHub query syntax. Workflows may select tasks using `is:issue label:chatgpt-task`.
