# Young-Consultations

Young-Consultations builds AI-assisted software-development and consulting tools.

## Current repositories

- `Young-Consultations/slugger` is the current primary product.
- `Young-Consultations/slugger-generated-demos` is a controlled generated-output sandbox for Slugger-produced draft pull requests.

Generated code requires human review before it is trusted, merged, or used beyond evaluation. Future repositories may include consulting operations and AI-SDLC components when those bodies of work have clear ownership and independent governance.

## Security

Do not report security issues publicly. Use private vulnerability reporting for the affected repository when available, or contact maintainers through a private organization-approved channel.
