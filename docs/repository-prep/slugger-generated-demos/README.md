# slugger-generated-demos

This is a Slugger-managed sandbox for generated Python CLI demos from `Young-Consultations/slugger`.

Generated projects arrive through draft pull requests. Generated output is untrusted until reviewed, generated applications are not automatically production-ready, and automatic merging is prohibited. Security and cryptography examples are educational unless independently reviewed. Maintainers should avoid manual changes to Slugger-owned generated branches. Serious projects should be promoted into independent repositories after review.
