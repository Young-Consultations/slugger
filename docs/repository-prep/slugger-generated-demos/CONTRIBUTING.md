# Contributing

This repository is a Slugger-managed sandbox. Generated projects arrive through draft pull requests and remain untrusted until human review is complete.

- Automatic merging is prohibited.
- Generated applications are not automatically production-ready.
- Security and cryptography examples are educational unless independently reviewed.
- Maintainers should avoid manual changes to Slugger-owned generated branches; request a new Slugger run instead.
- Serious projects should be promoted into independent repositories after review.
- Pull requests must record generated project name, source Slugger run, validation result, test result, smoke-test result, security review, dependency review, and promotion decision.
